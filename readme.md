# vakahata

(v)ast (k)ind aut(h)oriza(t)ion

vkht + a

Simple phalcon auth service.

## How to start?

You can start
on Ubuntu 22.04 with docker and docker-compose
from these steps:

- get vakahata-dir from repo
- go to vakahata-dir
- make env-file
- up vakahata-app
- test vakahata-app
- down vakahata-app
